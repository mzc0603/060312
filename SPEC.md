# 在线配色取色器 - 设计规格说明

## 1. Concept & Vision

一款专业级在线配色取色工具，融合现代极简美学与实用功能。界面如同一块精密的设计工具——干净、精准、优雅。让设计师和开发者能够高效完成色彩工作，无需任何多余干扰。

## 2. Design Language

### 美学方向
深色玻璃态设计（Dark Glassmorphism），参考 Figma/Pexels 等专业设计工具的视觉语言。深邃的背景衬托出色彩的鲜活，强调工具的专业感与使用时的沉浸体验。

### 配色方案
- **主背景**: `#0a0a0f` (深邃黑)
- **卡片背景**: `rgba(255, 255, 255, 0.05)` (毛玻璃)
- **边框色**: `rgba(255, 255, 255, 0.1)`
- **主强调色**: `#6366f1` (靛蓝紫)
- **次强调色**: `#22d3ee` (青色)
- **成功色**: `#10b981`
- **警告色**: `#f59e0b`
- **文字主色**: `#f8fafc`
- **文字次色**: `rgba(255, 255, 255, 0.6)`
- **文字弱色**: `rgba(255, 255, 255, 0.4)`

### 字体
- 主字体: `"Inter", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 等宽字体: `"JetBrains Mono", "Fira Code", monospace` (用于色值显示)

### 空间系统
- 基础单位: 4px
- 间距: 8px, 12px, 16px, 24px, 32px, 48px
- 圆角: 8px (小元素), 12px (卡片), 16px (大容器)
- 阴影: `0 8px 32px rgba(0, 0, 0, 0.3)`

### 动效哲学
- 所有过渡: 200-300ms ease-out
- 悬停反馈: 微妙上浮 + 光晕扩散
- 颜色变化: 平滑渐变 150ms
- 复制成功: 轻量弹跳动画

### 响应式策略
- **桌面 (>1024px)**: 双列布局，取色预览与颜色信息并排

- **手机 (<768px)**: 全宽设计，触摸优化，间距减小

## 4. Features & Interactions

### 核心功能

#### 4.1 图片上传取色
- 支持拖拽上传或点击选择
- 支持格式: JPG, PNG, WebP, GIF
- 上传后图片显示在取色预览区
- 鼠标悬停显示放大镜效果
- 点击图片任意位置获取该点颜色
- 支持键盘快捷键 `U` 上传图片

#### 4.2 屏幕吸管取色
- 点击"屏幕取色"按钮进入取色模式
- 使用 Canvas 截取屏幕
- 实时显示当前鼠标位置颜色
- 点击任意位置完成取色
- ESC 键退出取色模式
- 支持键盘快捷键 `S` 启动屏幕取色

#### 4.3 颜色值转换
- 实时显示: HEX, RGB, HSL, HSV, CMYK
- 点击任意格式可一键复制
- 颜色滑块: R/G/B 独立调节
- 色调/饱和度/亮度滑块
- 输入框直接编辑色值

#### 4.4 经典配色卡大全
- 内置 12+ 经典配色方案:
  - Material Design 色彩
  - 扁平化设计色板
  - 莫兰迪色系
  - 马卡龙色系
  - 复古色系
  - 科技蓝渐变
  - 森林绿系
  - 烈焰红系
  - 皇家紫系
  - 日系和风
  - 赛博朋克
  - 极简中性
- 每个配色卡展示 5-6 个颜色
- 点击配色卡中的颜色复制色值
- 双击配色卡复制整个配色

#### 4.5 收藏功能
- 点击星标按钮收藏当前颜色
- 收藏数据存储在 localStorage
- 我的调色板最多保存 20 个颜色
- 悬停显示颜色名称/色值
- 可删除单个收藏
- 支持清空全部

#### 4.6 一键复制
- 点击颜色值旁边的小图标复制
- 复制成功显示提示动画
- 支持 Toast 通知反馈

### 交互细节
- 所有可点击元素 hover 有视觉反馈
- 按钮点击有按压效果
- 输入框 focus 有边框高亮
- 滑块拖动实时更新颜色
- 图片加载显示 loading 状态
- 空状态有友好提示

## 5. Component Inventory

### 5.1 Header
- Logo: 渐变色图标 + "配色宝" 文字
- 暗色/亮色模式切换（默认深色）
- 响应式: 移动端 Logo 缩小

### 5.2 Color Preview Card
- 默认: 渐变背景 + 提示文字
- 有图片: 显示图片，带放大镜效果
- 取色中: 显示十字准星
- 状态: empty, loading, loaded, picking

### 5.3 Color Info Panel
- 色值显示区: 5种格式垂直排列
- 滑块区: RGB 滑块 + HSL 滑块
- 操作按钮: 复制、收藏、分享
- 状态: empty, valid

### 5.4 Action Bar
- 上传按钮: 图标 + 文字
- 屏幕取色按钮: 滴管图标
- 快捷键提示
- 状态: default, hover, active, disabled

### 5.5 Palette Card
- 配色名称 + 预览色块 (横向排列)
- 悬停: 整卡微上浮 + 显示复制提示
- 点击: 复制单个颜色
- 双击: 复制整个配色

### 5.6 Favorite Item
- 颜色色块 (圆角方形)
- 悬停: 显示 HEX 值 + 删除按钮
- 点击: 复制色值

### 5.7 Toast Notification
- 位置: 底部居中
- 类型: success (绿), info (蓝), error (红)
- 自动消失: 2秒

### 5.8 Color Picker Magnifier
- 圆形放大镜 (100px)
- 十字准星中心
- 当前颜色大字显示
- 像素网格效果

## 6. Technical Approach

### 技术栈
- 纯前端: HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，单文件实现
- ES6+ 语法
- CSS Grid + Flexbox 布局
- CSS Custom Properties 主题变量

### 核心实现
- **图片取色**: Canvas 2D `getImageData()`
- **屏幕取色**: `navigator.clipboard` + Canvas 截图
- **颜色转换**: 自实现 HEX↔RGB↔HSL↔HSV↔CMYK 算法
- **本地存储**: localStorage 存储收藏
- **SEO**: 完整 meta 标签、语义化 HTML、结构化数据

### 性能优化
- 图片懒加载预览
- 防抖处理滑块更新
- CSS will-change 优化动画
- 使用 CSS Variables 减少重复计算

### 浏览器兼容
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
- 移动端: iOS Safari 13+, Chrome Android 80+
